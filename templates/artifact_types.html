QIIME2 Artifact Types

Learn about different QIIME2 artifact types and their export formats

Back to Export Tool

FeatureTable

Feature tables store counts of features on a per-sample basis. These can be frequencies (counts) of amplicon sequence variants (ASVs), operational taxonomic units (OTUs), or taxonomic groups.

Export Format: BIOM v2.1 (.biom)

qiime tools export --input-path feature-table.qza --output-path exported-feature-table

FeatureData[Sequence]

Contains biological sequence data representing the features in a feature table.

Export Format: FASTA (.fasta)

qiime tools export --input-path rep-seqs.qza --output-path exported-rep-seqs

FeatureData[Taxonomy]

Contains taxonomic annotations for features in a feature table.

Export Format: TSV (.tsv)

qiime tools export --input-path taxonomy.qza --output-path exported-taxonomy

Phylogeny[Rooted]

Contains a rooted phylogenetic tree relating the features in a feature table.

Export Format: Newick (.tre)

qiime tools export --input-path rooted-tree.qza --output-path exported-tree

SampleData[AlphaDiversity]

Contains per-sample alpha diversity metrics.

Export Format: TSV (.tsv)

qiime tools export --input-path alpha-diversity.qza --output-path exported-alpha-diversity

PCoAResults

Contains principal coordinates analysis results.

Export Format: Multiple files including eigenvalues.txt and samples.txt

qiime tools export --input-path pcoa-results.qza --output-path exported-pcoa

DistanceMatrix

Contains distances between pairs of samples.

Export Format: TSV (.tsv)

qiime tools export --input-path distance-matrix.qza --output-path exported-distance-matrix

Visualization (.qzv)

QIIME 2 visualizations are exported as directories containing HTML files and supporting resources.

Export Format: HTML and supporting files

qiime tools export --input-path visualization.qzv --output-path exported-visualization

SampleData[PairedEndSequencesWithQuality]

Contains per-sample paired-end sequence data with quality information.

Export Format: FASTQ (.fastq.gz)

qiime tools export --input-path paired-end-demux.qza --output-path exported-paired-end-demux

Other Artifact Types

QIIME2 supports many other artifact types not listed here. This tool will automatically detect the artifact type and export it to the appropriate format.

For more information on QIIME2 artifact types, visit the QIIME2 documentation.
